AdvancedLayoutCalculator.TextProcessor.RawPiece — Type
RawPiece

Stores n-gram counts of a piece of text. Only accounts for raw characters.

julia> d = Dict(1 => Dict("a" => 10, "e" => 4));

to_pstring(::String)
